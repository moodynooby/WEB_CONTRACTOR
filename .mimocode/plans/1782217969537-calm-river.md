# PyQt6 → PyQt5 Migration Plan

## Goal
Replace PyQt6 with PyQt5 for Ubuntu 20.04 compatibility (Python 3.10+ via deadsnakes PPA). PyQt6 wheels require glibc/features not reliably available on 20.04.

## Summary of Changes
**9 files, ~56 discrete edits.** No structural changes — purely mechanical API mapping.

---

## Step 4: `closeEvent` signature (2 files)

PyQt6 passes `a0: QCloseEvent | None`. PyQt5 passes `event: QCloseEvent` (never None).

**`src/gui.py` line 262-267:**
```python
# Before:
def closeEvent(self, a0: QCloseEvent | None) -> None:
    self.task_manager.stop_all()
    self.log_poller_timer.stop()
    if a0:
        a0.accept()

# After:
def closeEvent(self, event: QCloseEvent) -> None:
    self.task_manager.stop_all()
    self.log_poller_timer.stop()
    event.accept()
```

**`src/ui/widgets/email_review_dialog.py` line 192-199:**
```python
# Before:
def closeEvent(self, a0: QCloseEvent | None) -> None:
    self._set_cards_enabled(False)
    self._clear_cards()
    if a0 is not None:
        a0.accept()

# After:
def closeEvent(self, event: QCloseEvent) -> None:
    self._set_cards_enabled(False)
    self._clear_cards()
    event.accept()
```

## Step 5: Enum flattening (5 files, ~35 enums)
